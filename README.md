# Gemini Clone – Cryptocurrency Trading Frontend
## Overview

**Gemini Clone** is a **frontend replica of the Gemini cryptocurrency platform**, built using **React JS**. The project integrates with the **Gemini API** to display real-time cryptocurrency prices and provide trading functionalities, giving users an interactive and responsive trading experience.

## Key Features

* **Real-Time Data:** Fetches live cryptocurrency prices and market data using Gemini API.
* **Trading Interface:** Simulated buy/sell functionalities for cryptocurrencies.
* **Responsive Design:** Works seamlessly across desktop and mobile devices.
* **User-Friendly UI:** Clean and intuitive interface inspired by the original Gemini platform.

## Technologies Used

* **Frontend:** React JS, CSS
* **API:** Gemini API
* **Other Tools:** VS Code, npm
